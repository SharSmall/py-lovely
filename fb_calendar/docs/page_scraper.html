Page Scraper¶

This module pulls pages that don’t already exist in the database when a user signs up that has liked those pages.

It then downloads the events upcoming for each of those pages; and creates a relationship between those events and the user that just signed up by creating UserEvent entries.

page_scraper.continuously_update(*args, **kwargs)[source]¶

Pulls all the user’s likes from the FB api and throw ‘em in the database. This guy loops every one second, grabbing all the users that just signed up.

This method has no return value, just the side effect that now the database will be full of that user’s pages and the associated events.

We don’t need to drill down into the user’s friends since they were at one point also all new active users.

Parameters:user_id (int) – The user’s fbid for which to pull likes.
