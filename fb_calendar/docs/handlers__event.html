handlers.event¶

Provides CRUD methods for Event objects. This module is also responsible for generating various feeds of events. There are a few in there now we’re just playing around with.

class handlers.event.UserEvents(application, request, **kwargs)[source]¶
get(*args, **kwargs)[source]¶

This endpoint is intended to provide a ranking of events as a list view. The corresponding detail view should be called Event. At the moment the default ranking is just by the number of people interested in the event. We will soon change this to a more sophisticated metric, but probably not for the MVP.

Parameters:
  • fbid (int) – The user’s Facebook ID
  • page (int) – Which page of results should be returned.
  • page_size (int) – The number of results to return per-page.
Returns:

A list of events by the default ranking as well as standard pagination parameters.

post(*args, **kwargs)[source]¶
This endpoint is intended to save an event to the user’s personal feed (the table user_event).
If the event already exists, only the updated_at field is modified.
Parameters:
  • s_user_id (string) – The user’s Facebook ID
  • s_event_id (string) – The ID of the event.
Returns:

A pair of user_id, event_id.
put(*args, **kwargs)[source]¶
This endpoint is intended to update fields on a list of events. The updates are specified
in the PUT body as below. id is required for each update.
[{"id": 0, "name": "my new event name"}, {"id": 1, "hidden": 1}, {"id": 2, "name": "my other new event name"}]

This endpoint is intended to provide a ranking of events as a list view. The corresponding detail view should be called Event. These events will be ordered by the number of people interested in them, and do not correspond to the user’s liked pages.

class handlers.event.CoolEvents(application, request, **kwargs)[source]¶
get(*args, **kwargs)[source]¶

This endpoint is intended to provide a ranking of events as a list view. The corresponding detail view should be called Event. These events are ranked by the ratio of those “going to” the event to those that have not yet replied to their invitation.
